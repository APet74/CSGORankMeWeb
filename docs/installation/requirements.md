# PHP Extensions
* PHP >= 7.1.3
* OpenSSL PHP Extension
* PDO PHP Extension
* Mbstring PHP Extension
* Tokenizer PHP Extension
* Ctype PHP Extension
* JSON PHP Extension 
* GMP PHP Extension

_You must also be using RankMe with MySQL and **NOT** SQLite!!_


**If all of these requirements are met you can start you installation process!**
